JSON Tools

The json tools is a local javascript based json operation tool, it supports query on multiple json object arrays using powerful SQL by assigning each array a unique identifier as table name.Specifically the SQL dialect used is Sqlite3,which is embedded in modern browser's javascript integration.

Design Purpose

The core essence of the JSON Tools is its miscellaneous functionalities provided through a simpler cleaner user interface.

Internal Reference

The JSON Tools uses SQLite as a sql server backend, so it requires the agent(i.e. the browser) to support Web SQL(which is deprecated now).The standard is here.

To avoid any possible limitation of depending on current deprecated API, we abstract a SQL client to handle sql operations and issues.
