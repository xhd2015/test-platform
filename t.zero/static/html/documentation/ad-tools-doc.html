Bean Introspection

The introspection capability is provided by the API: /internal/app/introspection/invoke.It can be used to invoke any registered bean and static method.It support a deep path resolution on a target object. Examples: 

localhost:8081/internal/app/introspection/invoke?beanClass=com.fulton_shaw.tools.core.service.experiment.DislikeMultiplyExperimentService&method=getDislikePolicy&args=%5B%22123%22%5D
