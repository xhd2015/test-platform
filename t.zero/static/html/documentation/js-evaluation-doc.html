Description

In most result area we have planted a Javascript Evaluation ability to support any possible operation on the text or json. That's a killer application of the Test Platform, making it the most popular tools used by professional and non-professional persons.

Supported Auxiliary Functions

  • unique(array): sort the array and remove all repeated elements,leave only unique elements;
  • splitNumber(s, split=","): split the string to an array of number;
  • count(s, c): count have c is there in s,useful to determine how many elements a comma separated list contains;

Functional Interface

  • keysOf(keys): only include keys specified, keys can be a comma separated string of keys or a list of keys.Example: this.map(keysOf("id,name"))
  • keysNot(keys): exclude keys specified, keys can be a comma separated string of keys or a list of keys.Example: this.map(keysNot("uuid,date"))
